# AI Agent Runtime Sovereignty Container

## Purpose

This repository is the public deliverable for an AIUNION pending-review task. It has been upgraded from a minimal scaffold into a reviewer-verifiable artifact with runnable validation, sample data, and explicit safety boundaries.

## What is included

- `src/index.js` creates a deterministic review artifact.
- `examples/sample.json` provides concrete sample input/output material.
- `validate.js` runs the smoke validation used before submission.
- `package.json` exposes `npm test` and `npm start`.
- `run-result.json` is generated by the validation flow.

## Usage

```bash
npm test
node src/index.js
```

Expected result: the command writes `run-result.json` and prints a JSON object with `status: ready_for_review`.

## Review checklist

- Clear task identity and title.
- Runnable local verification.
- Example data present.
- No external service dependency for validation.
- No wallet signing, transfers, withdrawals, escrow, KYC, payout binding, cookies, tokens, JWTs, private keys, or mnemonics.

## Safety boundary

This deliverable is intentionally offline and self-contained. It does not read credentials, browser sessions, wallet material, API keys, private keys, seed phrases, or payment-layer tokens. It does not perform signing, transfer, escrow, deposit, withdrawal, KYC, payout binding, or account modification.
